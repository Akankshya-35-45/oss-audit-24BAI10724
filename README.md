## Project Title
Open Source Audit using Linux Shell Scripts

## Description
This project demonstrates the use of Linux shell scripting to perform basic system-level operations.  
It includes scripts to check system information, verify Python installation, analyze disk usage, read log files, and take user input.

## Scripts Included

### 1. script1.sh – System Information
Displays:
- Current user
- Linux kernel version
- Current date and time

### 2. script2.sh – Python Check
Checks whether Python is installed in the system.

### 3. script3.sh – Disk & Permissions
Displays:
- Disk usage of /home directory
- Permissions of /home

### 4. script4.sh – Log Analyzer
Counts number of error messages in system log files.


### 5. script5.sh – User Input Script
Takes user input and creates a text file (manifesto.txt).
